Introduction

This vignette provides an overview and usage examples for the lab4 package, which implements multiple linear regression using ordinary least squares. The package includes a reference class linreg with methods to fit a linear regression model, access model statistics, and visualize the results.

Installation

To install the linreg package, you can use the following R code:

devtools::install_github("https://github.com/ductr388/lab4")

Getting Started

To begin, we need to load the lab4package package and create an instance of the linreg reference class that takes two arguments “formula” and “data”. For the inputs, “formula needs to be a formula-class and”data” needs to be a data frame-class with columns spelled the same as in the “formula”.
We will demonstrate the basic usage of the package with a sample dataset iris where Petal.Length is the response variable and Sepal.Width & Sepal.Length are the dependent variables.

library(lab4package)
model <- linreg$new(formula = Petal.Length ~ Sepal.Width + Sepal.Length, data = iris)

The linear regression is saved in model which will be used throughout the vignette.

Accessing Model Information

You can access various information about the model using the methods provided by the linreg class. For example, to access the model coefficients you can use:

model$coef()
##  (Intercept)  Sepal.Width Sepal.Length 
##    -2.524762    -1.338623     1.775593

If you also want to include the formula that the model is using you can use:

model$print()
## Call:
## linreg(formula = Petal.Length ~ Sepal.Width + Sepal.Length, data = iris)
## 
## Coefficients:
##  (Intercept)  Sepal.Width Sepal.Length 
##       -2.525       -1.339        1.776

If you want to print a summary of the model that includes the estimated coefficients, their standard errors, t-values, and p-values you can use summary(). Significance codes (***, **, *) are used to indicate the level of significance for each coefficient.

model$summary()
## Coefficients:
##              Estimate Std. error t value Pr(>|t|)    
## (Intercept)  -2.52476    0.56344  -4.481 1.48e-05 ***
## Sepal.Width  -1.33862    0.12236 -10.940 9.43e-21 ***
## Sepal.Length  1.77559    0.06441  27.569 5.85e-60 ***
## ---
## Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1
## 
## Residual standard error: 0.6465 on 147 degrees of freedom.

You can use the pred() method to obtain the predicted values from the regression model. These values represent the model’s predictions for each observation in the dataset.

head(model$pred())
##        1        2        3        4        5        6 
## 1.845579 2.159772 1.536929 1.493232 1.534157 1.842807

You can also use resid() method to access the residuals from the regression model. Residuals represent the differences between the observed values and the predicted values.

head(model$resid())
##            1            2            3            4            5            6 
## -0.445578965 -0.759772100 -0.236928933  0.006767993 -0.134157381 -0.142807413
